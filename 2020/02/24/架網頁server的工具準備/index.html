    這次架網站的server有用到的「東西」：

    ((說是東西其實就是別的大大寫好的、讓我們可以拿來減輕一堆負擔的code) 架一個server總之就是需要先選個語言，然後選一個合你語言的web框架，讓你寫程式可以直接叫人家寫好的函式庫(爽#
    然後因為我們寫了語錄生成器，算是(對使用者)互動式(?)，所以要有一個關聯式資料庫的管理系統，和一個幫忙生成最終文件的渲染引擎 這樣(#
    然後因為我還在用Windows系統，所以只講在Windows裡的安裝和使用




    語言我們選了Python，所以……

    web框架--cherryPy

    ......我們選了cherryPy((我都叫它櫻桃派

    cherryPy安裝

    先確認你家的python是不是介於3.5到3.8之間 是的話直接用command叫python的package manager幫你裝

    大部分的python裡都會有pip，所以建議大家直接用pip裝
    如果你同時還有裝python2的話，怕裝錯地方，可以
    pip3 install cherrypy
    這樣指明要裝在python3比較保險

    使用cherryPy

    可以先用command測試一下你的櫻桃派有沒有問題： $ python -m cherrypy.tutorial.tut01_helloworld 它會幫你開server聽訊號，然後指在你本地端的在127.0.0.1:8080，這是櫻桃派給的預設 如果你不想聽在8080的話要自己改 總之你下完指令之後它following都有出來的話基本上就是沒問題了 接下來就打開你家python編輯器，開頭import cherrypy 然後櫻桃派的doc乖乖研究一下，想用什麼自己用(有夠隨便

    資料庫管理系統--SQLite

    聽說這個系統效率沒有很好，所以如果你要寫流量比較大的網站大概就不太適合 不過因為我們對自己網站的低流量很有信心(欸)，所以就選了跟我一起寫這個網站的前輩沒玩過的SQLite 雖然這部分不是我寫的但我覺得看起來還算好懂好用啦(不負責任#

    SQLite安裝

    因為我們是用python寫server，所以這邊需要一個API接上python給的接口，我們選了apsw

    SQLite接python--apsw

    這裡雖然也可以用pip裝，不過我是直接去[這裡](https://rogerbinns.github.io/apsw/download.html) 下載符合自己windows和python版本的.exe檔 (愛放哪放哪 執行apsw，它就會幫你直接把SQLite的資料庫系統讀進記憶體

    使用SQLite

    可以直接先用command玩，試試看新增資料庫、顯示特定資料......之類的功能，大家自己去看docㄅ

    然後一樣記得在python編輯器內
    import apsw
    一樣把SQLite和apsw的文件讀一讀
    在你的server程式裡叫apsw.Connection(file.db)，就能用SQLite的函式庫了

    渲染引擎--Pyratem

    ((我叫它海盜##
    海盜它沒有LOGO(喔

    Pyratem安裝

    確認你家python版本新於2.6 (不過剛都已經說要介於3.5到3.8之間了wwww [下載](http://www.simple-is-better.org/template/pyratemp-0.3.2.tgz)

    使用Pyratem

    下載完解壓縮完，記得打開資料夾找到一個叫pyratem.py的檔案把它移到你專案的資料夾裡 一樣記得server的code裡import from pyratemp import Template 就可以在你的python code裡用模板了
